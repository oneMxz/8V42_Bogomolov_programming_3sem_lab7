# bayan

Утилита для обнаружения файлов-дубликатов по содержимому (сравнение по последовательности хешей блоков размера S).

## Запуск
bayan [options]

## Параметры
- --include-dir, -i <dir> [<dir> ...]
  Директории для сканирования (может быть несколько).

- --exclude-dir, -e <dir> [<dir> ...]
  Директории для исключения из сканирования (может быть несколько).

- --scan-level, -l <N>
  Уровень сканирования для всех директорий.
  0 — только указанная директория без вложенных.

- --min-size, -m <bytes>
  Минимальный размер файла (в байтах).
  По умолчанию 1 (т.е. проверяются файлы больше 1 байта).

- --mask <mask> [<mask> ...]
  Маски имён файлов, разрешённых для сравнения (не зависят от регистра),
  например: *.txt *.cpp *report*.*

- --block-size, -s <bytes>
  Размер блока S, которым производится чтение файлов.

- --hash <crc32|md5>
  Алгоритм хэширования H (crc32 или md5).

## Вывод
Полные пути файлов с идентичным содержимым:
- На одной строке один файл.
- Идентичные файлы идут одной группой.
- Разные группы разделяются пустой строкой.
